# 🚗 AI Powered Car Price Prediction System

Machine Learning web app that predicts car prices based on:

- Model
- Mileage
- Fuel Type
- Transmission
- Engine Size
- Tax
- MPG

## Tech Stack
- Python
- Pandas
- Scikit-Learn
- Streamlit
- Matplotlib
- Seaborn

## Features
- Real-time car price prediction
- Actual vs Predicted graph
- Correlation heatmap
- Price distribution analysis
- INR and GBP price conversion

## Run Locally

```bash
streamlit run app.py
